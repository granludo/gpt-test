tt1# gpt-translator
2 
3By Marc Alier forment @granludo https://wasabi.essi.upc.edu/ludo 
4 
5Licensed under the GNU General Public License v3.0
6 
7Simple python 3 program that uses OpenAi's GPT-3.5-turbo model to translate files from one language to another. Breaks down the file into chunks of 20 lines.
8 
9## requirement
10 
11You need to have a key to use the OpenAI API  you can get one here: https://beta.openai.com/docs/developer-quickstart/api-key  this program looks for the key in a file stated in the "mykeypath" variable by default on the path '..//..//mykey.json' 
12 
13You need to install the json, and openai packages
14 
15> pip install openai
16> pip install json
17 
18## usage 
19 
20> python3 gpt-translate.py origin_file origin_languaga destination_language destination_file
21 
22**Example**
23 
24> python3 gpt-translate.py sample.md catalan italian sample_translated.md 
25 
26Have fun.
